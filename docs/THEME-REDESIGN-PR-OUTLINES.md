# Theme Redesign PR Outlines

Use these as starting points when opening pull requests from the theme branches.

## `theme/minimal-luxury`

### Title

`Theme: Minimal Luxury Full-Site Redesign`

### Body

#### Summary

- Reframes the ExtraSure experience around a warm, premium, minimalist visual language.
- Preserves the existing application structure and product behavior while replacing the shared design system.

#### Highlights

- Refined neutral palette with understated contrast.
- More elegant typography pairing and calmer spacing rhythm.
- Premium treatment across homepage, CTA, account shell, and admin shell.

#### Validation

- `npm run build` pass
